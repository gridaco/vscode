## Publishing (for Bridged team)

Learn more at [publishing-extension](https://code.visualstudio.com/api/working-with-extensions/publishing-extension)

```sh
npm install -g vsce
vsce login bridged
vsce publish
# or with auto version up
vsce publish minor
```
